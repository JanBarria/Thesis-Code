# SO3 Quick Start Guide

## 🚀 Fast Track to Running the Experiment

### Prerequisites Checklist
- [ ] 2x PYNQ-Z2 boards with PYNQ OS
- [ ] Vivado 2020.1 or later installed
- [ ] PMOD A headers wired: Board1-Pin1→Board2-Pin2, Board1-Pin2→Board2-Pin1, GND connected
- [ ] Both boards connected to laptops via USB
- [ ] SSH access to both boards working

---

## Step 1: Build Bitstreams (30-60 minutes)

### Master Bitstream
```bash
cd SO3/scripts
vivado -mode batch -source create_project.tcl
# Wait for project creation, then:
vivado vivado_project/chaos_sync_master.xpr
# In Vivado GUI: Flow Navigator → Run Synthesis → Run Implementation → Generate Bitstream
```

### Slave Bitstream
1. Edit `hdl/chaos_sync_top.vhd` line 24-25:
   ```vhdl
   Y0_INIT : integer := 32768;  -- Change from 65536
   Z0_INIT : integer := 98304;  -- Change from 65536
   ```
2. Edit `scripts/create_project.tcl` line 14:
   ```tcl
   set project_name "chaos_sync_slave"
   ```
3. Repeat build process

---

## Step 2: Deploy to Boards (5 minutes)

```bash
# Master board
scp vivado_project/chaos_sync_master.runs/impl_1/system_wrapper.bit \
    xilinx@192.168.2.99:/home/xilinx/chaos_sync_master.bit
scp python/master_control.py xilinx@192.168.2.99:/home/xilinx/

# Slave board  
scp vivado_project/chaos_sync_slave.runs/impl_1/system_wrapper.bit \
    xilinx@192.168.3.99:/home/xilinx/chaos_sync_slave.bit
scp python/slave_control.py xilinx@192.168.3.99:/home/xilinx/
```

*(Replace IP addresses with your boards' actual IPs)*

---

## Step 3: Run Experiment (10 seconds + setup)

### Terminal 1 (Slave - START FIRST):
```bash
ssh xilinx@192.168.3.99
python3 slave_control.py
# Wait for "Waiting for MASTER to transmit..."
```

### Terminal 2 (Master - START SECOND):
```bash
ssh xilinx@192.168.2.99
python3 master_control.py
# Experiment runs for 10 seconds automatically
```

---

## Step 4: Analyze Results (1 minute)

```bash
# Retrieve data
scp xilinx@192.168.2.99:/home/xilinx/master_data.csv ./
scp xilinx@192.168.3.99:/home/xilinx/slave_data.csv ./

# Run analysis
cd SO3/python
python3 analyze_sync.py

# Check results
cat analysis_results/report.txt
```

**Success**: All correlation coefficients r ≥ 0.95 ✓

## Common Issues & Quick Fixes

| Problem | Solution |
|---------|----------|
| Slave receives no data | Check PMOD wiring (TX-RX crossover) |
| "No such file /dev/ttyPS1" | UART EMIO not enabled in Vivado |
| Low correlation (r < 0.95) | Verify different initial conditions in slave |
| Bitstream load fails | Check file path and permissions |
| AXI GPIO errors | Verify addresses in Python match Vivado |
